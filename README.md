# hwasungsi
FrontEnd + webSocket Chat  

#### 채팅 관련 소스코드
* static/css/chat_style.css
* static/js/chat.js
* templates/index.html


### Install
```
pip install flask flask_socketio eventlet pycryptodome
```

### Run
```
flask run
```

---
#### python version
* 3.11.4
#### 가상환경 설정
* conda activate 가상환경명
  ex) cozToy
