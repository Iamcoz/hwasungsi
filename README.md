# hs 프로젝트 FrontEnd + webSocket Chat  

#### 채팅 관련 소스코드
* static/css/chat_style.css
* static/js/chat.js
* templates/index.html


### Install 의존성 모듈 (requirements.txt)
```
pip install flask flask_socketio eventlet pycryptodome
```

### Run
```
flask run
```
-> http://localhost:5000

### Exit
command(ctrl) + c

---
#### python version
* 3.11.4
#### 가상환경 설정
* conda activate 가상환경명
  ex) cozToy
